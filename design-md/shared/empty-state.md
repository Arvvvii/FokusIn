# Shared Empty State — FokusIn

## Goal
Menampilkan keadaan kosong dengan visual yang tetap menarik.

## Empty States
- No discussions yet
- No materials uploaded
- No mentoring sessions
- No quiz available
- No notifications

## Layout
Centered illustration + text.

## Content
- Friendly message
- Short explanation
- CTA button

## Example
"Belum ada diskusi. Yuk buat pertanyaan pertama!"

## Style
- Cute minimal illustration
- Calm academic atmosphere
- Rounded cards
- Soft blue palette

## Important
Gunakan tone yang friendly dan encouraging.
