# Shared Loading State — FokusIn

## Goal
Loading state konsisten untuk seluruh aplikasi.

## Types
### Page Loading
Full page skeleton loading.

### Card Loading
Skeleton card shimmer.

### AI Loading
Animated AI processing state.

### Table Loading
Skeleton rows.

## Style
- Soft shimmer animation
- Rounded skeleton
- Calm blue tone
- Minimal motion

## Important
Gunakan loading state modern seperti Linear atau Notion.
