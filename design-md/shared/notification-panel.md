# Shared Notification Panel — FokusIn

## Goal
Panel notifikasi modern untuk aktivitas pengguna.

## Notification Types
- Forum replies
- Upvotes
- Badge earned
- Mentoring updates
- AI recommendation updates
- Material uploads

## Notification Card
- User avatar
- Notification icon
- Message preview
- Timestamp

## Actions
- Mark as read
- View detail
- Clear all

## Style
Modern floating notification panel.
Rounded card UI.
Soft shadow.

## Important
Gunakan icon berbeda untuk tiap tipe notifikasi.
